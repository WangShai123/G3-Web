# G3-Web

`G3-Web` 是一个 `WordPress`（以下简称 WP）主题框架。它以插件形式发布，为 WP 提供横向（系统应用层）和纵向（开发扩展层）的扩展。其目的是：

- 增强 WP 应用程序的能力，更好地满足现代数字化运营的需求。
- 帮助 WP 用户快速构建功能完善、稳定的 Web 项目，并轻松维护。
- 帮助中小创业者快速获取私有数据和应用系统，以超低成本实现跨平台运营和业务实验。
- 帮助任何有创意的用户将他们的想法转化为数字解决方案并快速实现。

## 项目路径

`/plugins/g3/`

## 特性

### 组件化

`G3-Web` 以组件化的方式扩展 `WordPress`，赋予其更多功能。然而，这些功能不是强制性的，也不会直接添加到 `WordPress` 中。

您可以通过选择所需的功能，将需要的功能添加到您的项目中。

### 多样化的接口

`G3-Web` 提供多种接口，包括：

- 函数（functions）
- 钩子（hooks）
- 短代码（shortcodes）
- wp-ajax
- rest-api

### 个性化

`G3-Web` 优先考虑用户隐私和数据安全，设计个性化的数据交互解决方案。您可以自定义 `WordPress` 应用程序的以下方面：

- 独立加密密钥
- 自定义管理页面 slug
- 自定义 `wordpress` 更新机制
- 自定义 API 接口命名空间地址
- 自定义 API 访问权限
- 自定义 API 访问频率
- 自定义重写路由规则
- 自定义用户 slug 规则
- 自定义文章浏览次数字段名称
- 自定义文章浏览次数更新频率
- 等等

### 免费教程

`G3-Web` 提供免费教程，帮助用户快速上手使用 `G3-Web` 并构建他们的项目。这些教程涵盖了广泛的主题。
