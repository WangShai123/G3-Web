# G3-Web

`G3-Web` 是一个现代化的 `WordPress` 插件框架，为 WP 提供横向（系统应用层）和纵向（开发扩展层）的扩展，旨在帮助开发者快速构建强大的 `WordPress` 主题和应用程序，快速服务业务：

- 增强 WP 应用程序的能力，更好地满足现代数字化运营的需求。
- 帮助 WP 用户快速构建功能完善、稳定的 Web 项目，并轻松维护。
- 帮助中小创业者以超低成本快速实现跨平台运营、业务实验和系统私有化。
- 帮助任何有创意的用户将他们的想法转化为数字解决方案。

## 🛜 项目路径

`wp-content/plugins/g3/`

## 🚀 核心特性

### 架构特性

- **现代化 PHP 8.3+** - 使用最新的 PHP 特性和语法
- **PSR-11 容器** - 符合 PSR-11 标准的依赖注入容器
- **组件化架构** - 模块化按需加载的组件系统，支持主题覆盖
- **RESTful API** - 基于注解的 REST API 路由系统
- **中间件支持** - 灵活的中间件机制，完善 API 权限控制
- **Rewrite** - 基于配置文件的重写系统
- **AOP 编程** - 面向切面编程支持

### 功能模块

- **队列系统** - 异步任务处理
- **缓存系统** - Redis 对象缓存支持
- **用户认证系统** - 支持微信登录、社交登录
- **内容管理** - 增强的文章管理、浏览统计、版权保护
- **电商功能** - 商品管理、订单系统、SKU 管理
- **微信集成** - 微信公众号管理、消息处理
- **安全防护** - 多层安全防护机制
- **支付系统** - 支持多种支付方式集成
- **等等**
